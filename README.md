⛏ MC PIXELATOR — Minecraft Pixel Art Generator

Перетвори будь-яке фото на піксель-арт у стилі Minecraft за секунди.
Без реєстрації, без серверів, без залежностей.


- Завантаження зображень — drag & drop або клік для вибору файлу
- 4 палітри блоків — Minecraft, Natural, Nether, Ender + автоматична палітра з вихідного фото
- Тонке налаштування — розмір пікселя, кількість кольорів, контраст, насиченість
- Ефекти рендеру — Shadow (об'єм блоків), Outline (контури), Dither (текстурний шум)
- Миттєве збереження результату у PNG одним кліком
- Адаптивна верстка — коректно працює на мобільних пристроях
- Minecraft-інтерфейс — зоряне небо, хмари, анімований логотип, піксельні шрифти


1. Завантажте зображення — перетягніть файл на зону завантаження або натисніть на неї
2. Виберіть палітру — Minecraft / Natural / Nether / Ender / Auto
3. Налаштуйте параметри:
   - Pixel Size — розмір одного блоку в пікселях (4–32)
   - Colors — кількість кольорів палітри
   - Contrast — посилення контрасту перед обробкою
   - Saturation — насиченість кольорів
4. Виберіть ефект — None / Shadow / Outline / Dither
5. Натисніть "CRAFT IT!" і завантажте результат


HTML5, CSS3, Vanilla JavaScript, Canvas API, FileReader API.
Зовнішніх бібліотек немає. Весь код — один файл, ~770 рядків.
